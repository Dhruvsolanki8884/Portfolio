## 🚀 Quick Start (3 Steps)

### Step 1: Install & Run
```bash
pnpm install
pnpm dev
```
Visit [http://localhost:3000](http://localhost:3000)

### Step 2: Customize (Edit `/lib/config.ts`)
- Update your information
- Add/modify projects
- Update skills

### Step 3: Deploy
```bash
vercel deploy --prod
```
Your portfolio is now live!

## 🔧 Customization Quick Reference

| What to Change | Where | How |
|---|---|---|
| Your name | `/lib/config.ts` | Update `portfolio.name` |
| Contact info | `/lib/config.ts` | Update `portfolio.contact` |
| Email in sidebar | `/components/sidebar.tsx` | Update email link |
| Profile photo | `/public/dhruv-photo.jpg` | Replace file |
| Projects | `/app/work/page.tsx` | Update projects array |
| Skills | `/lib/config.ts` | Update `portfolio.skills` |
| Social links | `/lib/config.ts` | Update `portfolio.social` |
| Colors | Component files | Replace `blue-600` with your color |
| Education | `/lib/config.ts` | Update education array |
| Experience | `/lib/config.ts` | Update experience array |
